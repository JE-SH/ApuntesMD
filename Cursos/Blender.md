# Blender

### Comandos

T, N -> ocultar barras

Botón scroll (BC), ctrl + BC, shift + BC -> Girar/Mover 

Shift + A -> agregar objeto

Tap -> modo edicion

G -> Grab para mover

R -> Rotate para rotar

S -> Scalar

shift + A -> add new object

shift + D -> Duplicate

X, Y, Z, al mover un objeto lo hace en esa dirección

Tab -> cambiar modo edición y objeto
